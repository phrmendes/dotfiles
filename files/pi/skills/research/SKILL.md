## Workflow

### Chatting with a paper

1. Read the PDF and extract text
2. For complex tables/equations, use `uv run docling` to parse the PDF to Markdown
3. Answer grounded in retrieved content, cite section/page

### Literature review

1. Search for relevant papers
2. Synthesize thematically, not paper-by-paper
3. Save notes in Markdown with Pandoc citation syntax `[@citekey]`

## Notes

- Always cite sources with paper title and section
- Write notes in Markdown
- For bulk operations (tagging, metadata, organizing), prefer agentic workflows
